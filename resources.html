Resources I found to be helpful for various subjects.

Self help

  • [link] The Academic Imperfectionist podcast by Rebecca Roache. Short, topical, well-supported podcasts.
    Free, self help, self care, productivity
  • [link] The Productive Researcher by Mark S. Reed. Focus on scientific research, but more generally applicable. Not free, but worth the money (starting from 5 pounds for a PDF).
    Productivity, research
  • [link] Blog post Lessons from my PhD by Austin Z. Henley.
    Free, productivity, presenting, supervisor interaction
  • [link] How to get your mojo back by Frank Martela. Online article about the issue of languishing, lacking motivation or exitement, but not really suffering.
    Free, self help, motivation

Tools

  • [link] "From Data to Viz leads you to the most appropriate graph for your data. It links to the code to build it and lists common caveats you should avoid."
    Free, visualisation
  • [link] Oh My Git! A game about learning git by bleeptrack and blinry.
    Free, git
  • [link] VisiData: Tabular data visualisation in the terminal.
    Free, visualisation, command line tool

Science

    Free, reviewing
  • [link] Twitter thread on manager interaction by Matthew Rechs.
    Free, supervisor interaction
  • [link] OpenAcademics writing resources.
    Free, writing
